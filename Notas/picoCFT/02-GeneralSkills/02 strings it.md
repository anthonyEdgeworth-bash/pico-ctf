# strings it

# Descripción

 Can you find the flag in file without running it?
# Solución


```  
picoCTF{5tRIng5_1T_1067EC4c}
```


```  
┌──(kali㉿kali)-[~/Downloads]
└─$ strings strings | grep pico
picoCTF{5tRIng5_1T_1067EC4c}
                                               
┌──(kali㉿kali)-[~/Downloads]
└─$ 

```
