# asm2

# Descripción

What does asm2(0x6,0x18) return? Submit the flag as a hexadecimal value (starting with '0x'). NOTE: Your submission for this question will NOT be in the normal flag format. Source
# Solución

```  
0x123
```

## Pasos para llegar a la solución

```  
- **Identificar Entradas**: Los argumentos son `0x6` y `0x18`.
    
- **Mapear Variables**:
    
    - `var1` (en `[ebp-0x4]`) comienza como `0x18`.
        
    - `var2` (en `[ebp-0x8]`) comienza como `0x6`
      
      Mientras `var2` sea menor o igual a `0xb518`, se ejecuta lo siguiente:

* A `var1` se le suma **1**.

* A `var2` se le suma **0xae** (174 en decimal).

4. **Calcular Iteraciones**:

* Para saber cuántas vueltas da el bucle:

$$\frac{0xb518 - 0x6}{0xae} \approx 266.4$$

* Como debe superar el límite, el bucle corre **267 veces**.

5. **Resultado Final**:

* `var1` final = `0x18` + 267 vueltas = 24 + 267 = **291**.

* Convertido a hexadecimal: **0x123**.
```
