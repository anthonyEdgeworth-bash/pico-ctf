# asm1

# Descripción

What does asm1(0x3ef) return? Submit the flag as a hexadecimal value (starting with '0x'). NOTE: Your submission for this question will NOT be in the normal flag format. Source
# Solución

```  
0x3e6
```

## Pasos para llegar a la solución

```  
1. Empezamos con `0x3ef`.
    
2. Como es menor que `0x6e6` y no es igual a `0x8`, saltamos a la resta.
    
3. Operación: `0x3ef - 0x9 = 0x3e6`
```
