# morse-code

# Descripción

Morse code is well known. Can you decrypt this? Download the file here. Wrap your answer with picoCTF{}, put underscores in place of pauses, and use all lowercase.
# Solución

```  
picoCTF{WH47_H47H_90D_W20U9H7}
```

## Pasos para llegar a la solución

```  
Subir el archivo de audio al siguiente enlance:
- https://morsecode.world/international/decoder/audio-decoder-adaptive.html
  
Decódificar el audio:

- W H47 H47H 90D W20U9H7
  
Darle formato de bandera:

picoCTF{WH47_H47H_90D_W20U9H7}

```
