
# Descripción
Can you convert the number 42 (base 10) to binary (base 2)?

# Solución

```  
picoCTF{101010} 
```

Para convertir de base 10 a  base 2 utilizamos la herramienta  [math.tools](https://math.tools/calculator/base/10-2), en ella podremos convertir los números
