# Descripción

If I told you a word started with 0x70 in hexadecimal, what would it start with in ASCII?



# Solución

[rapidtables.com](https://www.rapidtables.com/convert/number/hex-to-ascii.html "Pagina para convertir Hexadecimal a ASCII")
